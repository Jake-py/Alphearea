Как собрать статистику на сайте:

Админская панель
Создаёшь отдельный маршрут /admin (или /dashboard).
Ограничиваешь доступ только админам.

На странице выводишь:
Кол-во пользователей в базе (SELECT COUNT(*) FROM users;)
Кол-во пользователей онлайн (берём из Redis)
Кол-во заходов за период (логируем login/visit в PostgreSQL)
Кол-во просмотров страниц (по пингам или WebSocket)

Backend
API эндпоинты для панели:
/api/admin/users-count
/api/admin/online-users
/api/admin/visits

Возвращают JSON с цифрами.
Frontend
React / Vue / любой фреймворк:
Запросы к API
Красивые компоненты: числа, графики, диаграммы (Chart.js или Recharts)
Live-обновление через WebSocket или периодический fetch

Дополнительно
Фильтры по дате, типу пользователей, активности
Сортировка, графики, heatmaps
Можно добавить alert: “слишком много пользователей за 1 минуту” — пригодится при нагрузке
