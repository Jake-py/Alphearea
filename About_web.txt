Состояние сайта Alphearea:

Сайт представляет собой образовательную платформу с множеством функций для изучения языков и предметов. Вот подробное описание того, что работает и что еще не полностью функционирует.

Что работает:

Основная навигация и интерфейс:
- Главная страница с обзором доступных предметов
- Боковое меню для быстрого перехода между разделами
- Верхняя панель с заголовком и навигацией
- Responsive дизайн, адаптирующийся под разные экраны

Страницы предметов:
- Английский язык: курсы для начинающих, средний и продвинутый уровни, грамматика, словари (базовый, идиомы, фразовые глаголы), диалоги
- Корейский язык: курсы, диалоги, грамматика
- Русский язык: курсы, диалоги, грамматика
- Философия: книги, мудрость
- Психология: практики, теории
- Математика: материалы
- Программирование: материалы
- Электроника: материалы

Материалы и просмотр:
- Просмотр текстовых материалов по предметам
- Дерево материалов для навигации по структуре контента
- Умный просмотрщик материалов с возможностью чтения

Система тестирования:
- Создание тестов вручную (TestCreator)
- Прохождение тестов (TestTaking)
- Настройки тестов (TestSettings)
- Просмотр результатов тестов
- Управление тестами через API (создание, обновление, удаление)

Пользовательская система:
- Регистрация пользователей с валидацией данных
- Вход в систему с JWT токенами
- Изменение пароля
- Сброс пароля
- Профили пользователей с настройками
- Настройки аккаунта, приватности, сайта

Настройки и персонализация:
- Темная тема интерфейса
- Настройки языка
- Управление профилем пользователя
- Приватность и настройки сайта

API и бэкенд:
- Полная система аутентификации
- Хранение пользователей и профилей в JSON
- Управление тестами через REST API
- Хэширование паролей с Argon2
- CORS поддержка для фронтенда

Что не работает или работает частично:

ИИ-функции:
- Чат с ИИ-ассистентом JARVIS (требует запущенного Ollama с моделью gemma2:2b)
- Генерация тестов через ИИ (TDS система)
- Автоматическая обработка и парсинг PDF и DOCX файлов
- Умный редактор материалов (SmartEditor) - интерфейс есть, но ИИ-функции не работают

Отслеживание прогресса:
- Базовое отслеживание прогресса по предметам (уровни beginner/intermediate/advanced/expert)
- Логирование просмотра материалов
- Система опыта (XP) и уровней
- История тестов и результатов
- Но полная интеграция и визуализация прогресса не завершена

Система достижений:
- Структура для достижений есть в профилях
- Автоматическое присвоение достижений за уровни и тесты
- Но отображение достижений в интерфейсе не реализовано полностью

TDS система тестирования:
- Интерфейс для TDS (Too Damn Smart) тестирования
- Три режима работы (стандартный, адаптивный, глубокий анализ)
- Но ИИ для адаптивного тестирования не работает без Ollama

Парсинг материалов:
- Библиотеки для обработки PDF, DOCX, TXT установлены
- Утилита scanMaterials.js для сканирования
- Но автоматический парсинг и структурирование материалов не протестирован

Дополнительные материалы:
- Некоторые разделы имеют материалы (тексты, диалоги)
- Но загрузка и управление материалами через интерфейс не полностью реализовано
