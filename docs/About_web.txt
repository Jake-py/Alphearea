1. ОСНОВНАЯ ИНФОРМАЦИЯ О САЙТЕ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Alphearea - это мультиязычная образовательная платформа, предназначенная для обучения
различным предметам и языкам. Сайт поддерживает регистрацию пользователей, систему 
профилей, отслеживание прогресса, прохождение тестов и новую систему игровых очков 
(points) за достижения.

АРХИТЕКТУРА:
  • Frontend: React + Vite (localhost:5173)
  • Backend: Node.js/Express (localhost:3002)
  • Хранилище данных: JSON файлы (profiles, accounts, tests)
  • Аутентификация: JWT токены
  • Безопасность: Argon2 хэширование паролей, Content Security Policy

🎮 СИСТЕМА ИГРОВЫХ ОЧКОВ (Points System)
───────────────────────────────────────

Введена полная система начисления и использования игровых очков для мотивации 
обучения:

  📍 Источники получения очков:
     • +10 points за завершение каждого урока
     • +50 points за успешное прохождение теста (≥70%)
     • Бонусы за достижения и уровни
     • Специальные события и квесты

  📊 Счетчик Points в Header:
     • Отображение текущих очков в верхней панели
     • Оптимизирован для полного видимости
     • Обновляется в реальном времени при получении очков
     • Всплывающее окно с анимацией награды

  💾 Система хранения:
     • Локальное хранилище (localStorage) для кэширования
     • Backend персистентность (JSON файлы с профилями)
     • История всех транзакций очков
     • Защита от "фарма" (повторного прохождения)

5. КАК ПОЛУЧИТЬ ОЧКИ (Points)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

СПОСОБ 1: Завершение уроков
  1. Перейти на страницу курса (например, /english/grammar)
  2. Нажать кнопку "✓ Завершить урок"
  3. Получить +10 points
  4. Видеть анимированное окно награды

СПОСОБ 2: Прохождение тестов
  1. На странице курса нажать "🚀 Начать тест"
  2. Ответить на 10 вопросов за 20 минут
  3. Получить результаты
  4. Если результат ≥ 70%:
     • +50 points добавляются автоматически
     • Показывается поздравительное окно
     • История сохраняется на backend

ПРИМЕЧАНИЕ: Каждый тест можно пройти только один раз для защиты от фарма.
Очки хранятся в профиле и синхронизируются между устройствами.

Q: Как увеличить количество тестов?
A: Добавьте новые компоненты тестов в src/pages/ и создайте маршруты в App.jsx

Q: Можно ли изменить количество очков за тест?
A: Да, отредактируйте src/config/pointsService.js функцию awardPoints()

Q: Как добавить новый язык/предмет?
A: Создайте новую папку в src/data/materials/ и добавьте маршрут в App.jsx

Q: Как сбросить очки пользователя?
A: Отредактируйте JSON файл профиля в back-end/profiles/

Q: Может ли пользователь потратить очки?
A: Да, используется API endpoint POST /api/profile/use-points

Q: Как работает защита от фарма?
A: PointsTracker в localStorage отслеживает пройденные тесты
